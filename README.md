# 🤖 AI Daily

每日 AI & 科技资讯聚合，自动推送到 Telegram。

## 商业模式

### 免费版
- 每天自动推送中文 AI 资讯摘要
- 来源：Hacker News、TechCrunch AI、MIT Tech Review 等

### 付费版 ¥10/月
- 英文版资讯推送
- 关键词监控（自定义提醒）
- 历史搜索
- 多语言支持

## 项目结构

```
ai-daily/
├── scraper.py          # 新闻抓取
├── notifier.py         # 推送通知
├── bot.py              # Telegram 订阅机器人
├── subscribers.json    # 订阅者数据库
├── requirements.txt    # Python 依赖
└── .github/workflows/  # 每日定时任务
```

## 盈利关键指标

- 目标：100 付费订阅者
- 月收入目标：¥1,000
- 获客渠道：Telegram 搜索、知乎、Twitter
